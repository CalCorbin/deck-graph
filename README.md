# ✨ `Getting Started →`

## `📦 [Step 1] Environment Setup`
- Install →
    ```bash
    # Run npm i
    cd ~/code/deck-graph/
    npm i
   
    # Verify setup went smoothly
    npm run test
    ```

##  `⚡ [Step 2] Running the Server`
- Run →
   ```bash
   # Start local dev env
   npm run dev
   
   # Open the browser
   open http://localhost:3000/
   ```
